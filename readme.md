## Run Locally

- Clone the project

```bash
git clone https://github.com/Alexander-Shulginov/V-Notes
```

- Go to the project directory

```bash
cd V-Notes
```

- Install dependencies

```bash 
npm i
```

- Start the server

```bash 
npm run dev
```

## Hotkeys
`Ctrl + s` - toggle sidebar  
`Ctrl + c` - toggle color theme
`Ctrl + m` - toggle menu
